# Статистика переводов

Страница статистики даёт актуальный обзор покрытия переводов по всем установленным языкам.

```blade
<a href="{{ route('lingua.statistics') }}">Статистика переводов</a>
```

<Screenshot src="/screenshots/statistics-page.png" alt="Страница статистики переводов" caption="Страница статистики показывает покрытие по языкам, разбивку по группам и отсутствующие ключи." />

## Обзор

В верхней части страницы отображаются три сводных счётчика:

- **Всего ключей** — общее количество строк перевода в базе данных
- **Всего групп** — количество различных групп переводов
- **Установленные языки** — количество языков, зарегистрированных в системе

Вы можете включить **Включить переводы вендоров**, чтобы добавить или исключить переводы пакетов из всей статистики.

## Покрытие по языкам

<Screenshot src="/screenshots/statistics-coverage.png" alt="Раздел покрытия по языкам" caption="Каждая строка языка показывает полосу прогресса, процент и количество отсутствующих ключей." />

Для каждого установленного языка вы можете видеть:

| Столбец | Описание |
|---|---|
| Язык | Название и флаг языка |
| Покрытие | Полоса прогресса с процентом завершения |
| Переведено | Количество ключей, имеющих значение для этой локали |
| Отсутствует | Количество ключей без значения — нажмите для раскрытия панели отсутствующих ключей |

**Язык по умолчанию** выделен бейджем. Поскольку локаль по умолчанию определяет эталонный набор ключей, она всегда показывает 100% покрытие.

## Детализация отсутствующих ключей

Нажмите на **бейдж количества отсутствующих** в любой строке языка, чтобы развернуть панель отсутствующих ключей прямо в строке.

<Screenshot src="/screenshots/statistics-missing.png" alt="Развёрнутая панель отсутствующих ключей" caption="Панель отсутствующих ключей показывает группу, ключ и прямую ссылку на редактор переводов." />

Каждая строка панели показывает:

- **Группа** — группа перевода (например, `validation`, `auth`)
- **Ключ** — ключ перевода (например, `required`, `failed`)
- **Перевести →** ссылка — открывает страницу переводов с предфильтрацией по этой локали

Нажмите на ту же строку языка ещё раз, чтобы свернуть панель.

## Разбивка по группам

<Screenshot src="/screenshots/statistics-breakdown.png" alt="Таблица разбивки по группам" caption="Таблица разбивки показывает количество переведённых ключей по локалям для каждой группы." />

Таблица разбивки перечисляет каждую группу переводов с количеством переведённых ключей по каждому установленному языку. Используйте её для определения, какие группы полностью переведены, а какие требуют внимания.

## Переключение переводов вендоров

По умолчанию переводы вендоров (из пакетов вроде `laravel/framework`) исключены из статистики. Включите **Включить переводы вендоров**, чтобы добавить их.

::: info
Переводы вендоров часто поддерживаются автором пакета. Включение их в статистику может снизить видимое покрытие, если вы не добавили переопределения для конкретных локалей.
:::

::: tip Эффективная работа
Сортируйте по количеству отсутствующих ключей, чтобы расставить приоритеты в работе с переводами. Языки с наибольшим количеством отсутствующих ключей требуют наибольшего внимания.
:::
