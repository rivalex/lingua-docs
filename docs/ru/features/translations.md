# Управление переводами

Страница «Переводы» (`/lingua/translations/{locale?}`) позволяет просматривать, фильтровать и редактировать все строки переводов.

<Screenshot src="/screenshots/translations-page.png" alt="Страница переводов Lingua" caption="Страница переводов — с переключателем локалей, фильтром по группе и встроенным редактором." />

## Навигация по переводам

### Переключатель локалей

Вкладки локалей вверху страницы позволяют переключаться между установленными языками. URL обновляется до `/lingua/translations/{locale}`, поэтому каждый вид можно добавить в закладки и поделиться им.

Колонка **локали по умолчанию** всегда отображается слева в качестве справки — вы редактируете выбранную локаль справа.

### Фильтрация

| Фильтр | Описание |
|---|---|
| **Поиск** | Ищет по названию группы, ключу и всем значениям локалей |
| **Группа** | Фильтрует по конкретной группе (например, `auth`, `validation`, `single`) |
| **Тип** | Фильтрует по `text`, `html` или `markdown` |
| **Только непереведённые** | Показывает только строки без значения в выбранной локали |

<Screenshot src="/screenshots/translations-filters.png" alt="Фильтры переводов" caption="Фильтрация для отображения только отсутствующих французских переводов в группе validation." />

### Пагинация

Результаты разбиты на страницы с настраиваемым количеством записей на страницу (25 / 50 / 100). Текущая страница и количество записей сохраняются в строке запроса URL.

## Редактирование переводов

### Встроенное редактирование

Нажмите прямо на ячейку перевода, чтобы начать редактирование. Изменения сохраняются при потере фокуса (когда вы кликаете в другое место или нажимаете Tab).

- Переводы типа **plain text** используют простой `<textarea>`
- Переводы типа **HTML** открывают rich-text редактор TipTap
- Переводы типа **Markdown** открывают markdown-редактор TipTap

<Screenshot src="/screenshots/translation-editor-html.png" alt="HTML-редактор переводов" caption="Редактор TipTap HTML для переводов с форматированием." width="512px" :center="true"/>

### Синхронизация из основного языка

У каждой строки перевода есть кнопка **Sync from default** (↺). Нажатие на неё копирует значение локали по умолчанию в текущую локаль — полезно как отправная точка, когда нужны лишь незначительные изменения формулировок.

### Модальное окно редактирования

Для локали по умолчанию нажмите иконку карандаша, чтобы открыть модальное окно редактирования, где можно изменить тип перевода (text / html / markdown), а также значение.

::: tip Изменение типов переводов
Если вы меняете тип перевода с `text` на `html`, встроенный редактор в строке немедленно переключится на TipTap. Хранимое значение не изменяется — меняется только редактор.
:::

## Создание переводов

Нажмите **New Translation** для создания пользовательской записи. Заполните:

- **Group** — эквивалент имени файла (например, `marketing`, `emails`)
- **Key** — ключ внутри группы (например, `hero_title`)
- **Type** — `text`, `html` или `markdown`
- **Value** — перевод для локали по умолчанию

```
group: marketing
key:   hero_title
type:  text
value: Build faster, together.
```

Новый ключ сразу становится доступен через `__('marketing.hero_title')`.

::: warning Переводы пакетов
Вы не можете создавать переводы непосредственно в группе пакета (например, `validation`, `auth`). Эти группы принадлежат Laravel или другим пакетам и синхронизируются автоматически. Если вы хотите переопределить строку пакета, отредактируйте её напрямую на странице переводов.
:::

## Удаление переводов

### Удаление для конкретной локали

Для любой не-основной локали кнопка удаления убирает только значение этой локали из JSON-столбца. Ключ перевода продолжает существовать; он просто возвращается к локали по умолчанию.

### Полное удаление

В представлении локали по умолчанию кнопка удаления удаляет всю строку из `language_lines`. Используйте это для очистки ключей, которые больше не используются в вашей кодовой базе.

::: danger Защита переводов пакетов
Переводы пакетов нельзя удалить. Попытка показывает предупреждение и вызывает событие `vendor_translation_protected`. Вы можете только **редактировать** значения переводов пакетов.
:::

## Копирование ключа в буфер обмена

У каждой строки есть иконка буфера обмена, которая копирует полную ссылку `group.key` (например, `auth.failed`) в буфер — удобно при ссылке на ключи в Blade или PHP.

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Tab` | Сохранить текущее поле и перейти к следующему |
| `Shift + Tab` | Сохранить текущее поле и перейти к предыдущему |
| `Escape` | Отменить изменения и закрыть редактор |
