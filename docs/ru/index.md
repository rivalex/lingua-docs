    title: 70+ языков
    details: Установите любую локаль из экосистемы Laravel Lang одной Artisan-командой. Языковые файлы, записи в БД и переводы настраиваются автоматически.

  - icon: ✍️
    title: Поддержка rich text
    details: Каждый перевод может быть простым текстом, HTML или Markdown. Нужный редактор активируется автоматически в зависимости от типа перевода.

  - icon: ↔️
    title: Поддержка RTL / LTR
    details: Арабский, иврит, персидский и другие RTL-языки полностью поддерживаются. Lingua хранит и предоставляет направление текста для каждого языка.

  - icon: 📦
    title: Переводы пакетов (vendor)
    details: Управляйте переводами сторонних пакетов (сообщения валидации, пагинация и др.) наравне со своими строками — со встроенной защитой от случайного удаления.

  - icon: 📊
    title: Отслеживание прогресса
    details: Для каждого языка отображается процент заполнения и количество непереведённых строк — вы всегда знаете, что ещё нужно перевести.

  - icon: 🧪
    title: Полное покрытие тестами
    details: Более 150 Pest-тестов, охватывающих все Artisan-команды, Livewire-компоненты, Blade-компоненты, хелперы и полный API фасада.
